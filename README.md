# 2D Action Rogue-Lite (Prototype)

C++ の文法習得とゲーム制作経験を目的とした  
**横スクロールアクション × ローグライク（1ステージ）** の小規模ゲームプロジェクトです。

開発期間は **8週間** を想定し、社会人でも継続できるよう  
毎週 “1つの小さな成果” を積み上げる方針で進行します。

---

## 🎮 Game Concept

- シンプルな横スクロールアクション
- 毎回少しだけ変化する軽いローグライク要素  
  （敵配置やアイテム位置のランダム化）
- 1ステージのみで完結
- 「動くものを作る」ことを最優先にしたミニマムな構成

---

## 🧑‍💻 Purpose

このプロジェクトの目的は以下の通りです。

- C++ の文法・構造の習得  
- 2Dゲーム制作の流れを体験する  
- 小規模でも“完成する”プロダクトを作る  
- 3人で協力しながら開発経験を積む

---

## 🛠️ Technology Stack

- **Language:** C++17 以上  
- **Library:** （選択予定）SFML / SDL2 / Raylib  
- **Build:** CMake  
- **IDE:** VSCode / CLion（メンバーによって自由）  
- **Version Control:** Git + GitHub

### Week 1：環境構築
- C++環境を揃える

- パトロールする敵1体

- HPバー

### Week 7：調整
- 操作性調整  
- バグ修正

### Week 8：仕上げ
- 最終ビルド作成  
- 動作確認  
- 簡易ドキュメント化
